Project Title: Smart Agriculture Recommendation System using Machine Learning

Description:
Traditional farming often relies on intuition, leading to unpredictable yields and soil degradation. This project introduces a Smart Agriculture AI Platform that leverages the Random Forest algorithm to provide scientific farming advice.

The system operates in two modes:

Soil Analysis Mode: Takes precise inputs (Nitrogen, Phosphorus, Potassium, pH, Rainfall, etc.) to predict the optimal crop, expected yield (in Quintals/Acre), and specific fertilizer mix.

Historical Trend Mode: Offers recommendations based on crop rotation logic and regional history for farmers lacking immediate soil testing data.

Built with a Python Flask backend and a responsive HTML/JS frontend, the platform is designed for accessibility. It features a Voice Assistant for hands-free operation, a Rule-Based Chatbot for agronomy queries, and Multi-Language Support to bridge the language barrier for rural farmers. The system is robust, featuring an automatic "Demo Mode" fallback to ensure usability even during server downtimes.
